# Binary tree

В случая трябва да имплементирате дърво, което описва някаква логика по ваш избор. Например файлова директория или каквото друго решите, но трябва да кажете каква е целта ви.
Трябва да покажете основните операции (ако са приложими):

- добавяне
- премахване
- търсене
- обхождане (DFS, BFS)

Като цяло това е доста свободна задача и може да се застреляш лесно в крака, ако решиш да имплементираш нещо прекалено сложно.

Стандартен интерфейс:

```c++
class BinaryTree {
public:
 bool contains(const T&) const;
 void add(const T&);
 void remove(const T&); 

  // пишете обходането, за което имате необходимата структура, за да го реализирате, не си спомням дали можеше да се ползват std::queue за bfs
  // dfs може с рекурсия, не е нужен std::stack
 // inorder, preorder or postorder
 void dfs() const;
 void bfs() const;
};
```
